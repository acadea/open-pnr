# Open PNR

An Open Source Amadeus PNR converter.


## Installation

Install the package

```bash
npm i open-pnr
```

## Usage
```js
import {pnrParser} from 'open-pnr';
// .... 
const parsed = pnrParser(pnrText);
console.log(parsed);
```
